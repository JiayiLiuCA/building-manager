如何开始对接

请阅读该文档了解开放平台、对接流程（通通锁App对接，通通酒店App对接），小程序插件，API, SDK, 常见问答等。

  • 对接通通锁App，设备添加至通通锁App；请参考章节一、二；

  • 对接通通酒店App以及发卡器，设备添加至通通酒店App；请直接参考章节三：通通酒店对接；

  • 根据SDK（或小程序插件）开发自己的平台，请参考：App集成开发指南；

注意：关于对接，我们不提供测试账号或沙箱环境，对接直接参考对应的对接流程即可

一、开发者账号和应用

1.1、注册开发者账号

  • 开发者账户仅用于申请应用，不用于其他用途。开发者账户和通通锁App账户不同，二者也不用相关联；

  • 关于开放平台的相关问题，如是否收费，如何升级等，请参考文档：开放平台相关问题

1.2、开发者账户中创建应用并申请，审核通过会获取client id & secret

  • 应用创建后默认是审核中状态，后台会在工作日内由人工审核，节假日会有延迟，请耐心等待。

  • 开发者可以在审核通过的应用详情页查看client_id和client_secret.

  • 如果有多个项目，一个开发者账号可以创建多个应用。

二、对接通通锁App

2.1、在通通锁APP里添加锁

  • 在应用市场搜索“通通锁”，下载并安装App，注册用户账号，并添加锁。

  • 通通锁使用手册：App、Web

2.2、获取通通锁App账户的访问令牌

  • 用2.1中注册的通通锁App账号和密码 + 1.2中申请的应用信息（client_id 和 client_secret ）调用云端接口, 获取添加锁账户的访问令牌（accessToken)

注意accessToken会在expires_in标识的时间后失效，默认为90天，失效后需要重新获取或刷新

2.3、获取accessToken之后，可以用其调用其他API

  • 详细图文步骤请参考链接：如何对接通通锁App

三、通通酒店对接

通通酒店对接不需要开放平台注册开发者账户以及申请应用，可以直接去通通酒店电脑客户端--设置--门锁对接(第一次需先申请对接)--获取对接信息

对接信息包含client_id, client_secret, 账户信息和密码，用对接信息直接调用API即可，详细图文步骤请参考链接：通通酒店对接、发卡器对接

通通酒店App使用手册：介绍 | 通通酒店帮助中心

四、API

提供锁、密码、电子钥匙、IC卡、指纹、网关等接口，具体接口文档请参考开放平台 Cloud API v3 部分；

开放平台不仅仅只包含了下面介绍的API，其他API可以自行探索开放平台文档Cloud API v3 部分

  • 密码开锁：支持限期密码，单次密码，自定义密码（自定义密码要求设备在线，即锁连接了网关 或 锁是WiFi锁）等多种密码类型；

  • 蓝牙开锁：也称为电子钥匙(eKey), 收到电子钥匙的用户可以用手机蓝牙进行开锁，一般需要下载App；

  • 远程开锁：需要设备联网（锁连接了网关 或者 WiFi锁），可以调用云端API进行远程开锁/闭锁等；

  • IC卡开锁：蓝牙录入IC卡 或者 调用API远程下发IC卡，通过IC卡片进行开锁；

  • 指纹开锁：通过录入指纹进行开锁；

  • 人脸识别：通过录入人脸 或 下发人脸照片进行开锁；

  • 开锁记录：上传、获取开锁记录，或回调通知；

五、SDK

SDK只是进行蓝牙离线操作；如果需要数据同步至云端，在调用SDK方法之后，需调用章节四中对应的API进行同步

如不需要云端同步（只想蓝牙离线控制），可以不调用非必须的API

  • IOS：IOS SDK & Demo

  • Android：Android SDK & Demo

  • Flutter ：Flutter SDK & Demo

六、小程序插件

小程序插件只是进行蓝牙离线操作；如果需要数据同步至云端，在调用SDK方法之后，需调用章节四中对应的API进行同步

如不需要云端同步（只想蓝牙离线控制），可以不调用非必须的API

  • 微信小程序：通通锁对接、水电表对接

  • 支付宝小程序：通通锁对接

七、私有化（本地化）部署（需付费）

如果希望：

  • 所有数据都存放在第三方软件自己的服务器上。

  • 密码和电子钥匙也是通过第三方软件自己的服务器生成。

  • 所有操作都不需要经过通通锁服务器，所有的数据都不存放在通通锁服务器。

  • 网关、API都连接本地服务器，实现内网需求。

可以进行本地化部署，请参考文档：本地化部署
